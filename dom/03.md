# `HTML*Element`개체 개요

HTML 문서 내 각 element들은 고유한 성질을 지니고 각 elt를 DOM 트리내의 노드 개체로 인스턴스화하는 고유한 JS생성자를 가진다.

ex) `<a> element`는 `HTMLAnchorElement()`생성자를 통해 DOM 노드로 만들어짐.
